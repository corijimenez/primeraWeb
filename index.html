HTML Hypertext Mackup Language

HTML is the markup language used to create the structure and content of a webpage. It is made up of elements, tags, and attributes that define the layout and hierarcy of content.

  • Header: Defined using the <head> <head> tag. It includes elements like the page title, metadata (such as character encodig and the page description), and links to CSS stylesheets and other resources.
  • Body: Defined using the <body> tag. This is where all visible content on the page goes, such as headers, paragraphs, images, links, and other elements.
  • Footer: Defined using the <footer> tag. This is a container that usually includes secondary or supplementary information about the site, like contact info, additional links, credits, etc.

CSS Cascading Style Sheets

CSS is the language used to five style and desing to a webpage. It controls the appearance of HTML elements, such as colors, fonts, size, margins, and positioning.

  1. Selectors: CSS selectors identify the HTML elemnts to which the styles will be applied. They can be tag selectors (like h1, p, img), class selectors (like .header, .container), or ID selectors (like #menu, #footer).
  2. Properties: CSS propierties define how the selected elements will look. For example, color sets the text color, font-family sets the font, margin sets the margin, and background sets the background color.
  3. Values: Values are assigned to tropoerties and determine how the elements will appear. For example, the values blue for coor sets the text to blue, and 20px for font-size sets the text size to 20 pixels.
